# Product

Acme Ops is an internal automation workspace for finance and operations teams that repeatedly transform Excel workbooks into reviewed exports.

## Users

Operators create reusable flows, start runs from those flows, upload workbooks, approve AI-generated sheet plans, and provide human inputs such as zip passwords.

## Product Purpose

The app should make workflow automation feel concrete: a flow is a graph of tool-backed steps, a run is an execution of that graph, and every paused step explains what the user must do next.

## Tone

Calm, precise, agentic, and task-focused. AI should feel like an inline collaborator that edits the workflow structure, not a decorative chatbot beside the product.

## Strategic Principles

- The step graph is the primary object.
- Tools are first-class contracts for steps.
- Human input pauses must be obvious and actionable.
- Flow library, Composer, Runs, Tools, and Insights should use predictable navigation and consistent component vocabulary.
- Avoid decorative AI styling, generic dashboard cards, and unclear workflow terminology.
